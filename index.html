プロジェクト課題について

プロジェクト課題ではWWW のサーバへ検索し，サーバから届いたデータを Web ページとして整形するような Web アプリケーションを作成し、 グルメ情報についてのデータを検索、結果表示できるように作る。1回で終わるものではなく、合計6回にわたる講義で、課題を完成させる。ファイル名はproject.htmlである。 グルメ情報における検索キーはgenreの1つだけである。検索結果のデータ項目はアクセスや住所、予算などのお店に関して詳しくわかるものを表示する。
