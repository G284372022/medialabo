第1回 WWW と HTML

今回はHTMLの使い方について学んだ。 HTMLはWebページの構造を記述するための書式であり。HTML文書を作ることによって、Webページに表示するコンテンツと、ページのレイアウト作ることができるようになる。 要素にはブロック要素とインライン要素があり、インライン要素の中にブロック要素は配置できない。また要素には属性という付加情報をつけることができる。 属性にはclass,id,imgがあり、これはCSSなどでページを装飾するときに、見た目を柔軟に変更するために使用したりすることができる。

HTMLを使うためのお約束や要素を学んだ。HTMLの使い方を調べることが可能になった。そしてデバックが簡単にできる方法を学んだ。 また「www」のことや、URLのちょっとした仕組みについても学んだ。加えて、Githubの使い方について復習した。また次の課題に取り組んだ。

  • 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回はCSSを利用し、Webページを装飾する方法について学んだについて学んだ。 CSSでHTMLを装飾する方法は3つあり。今回は外部スタイルを利用した。

CSSはセレクタ・プロパティ名・プロパティ値でできている。セレクタで装飾したい要素、プロパティ名で種類、プロパティ値で数値を設定できる。 プロパティ値には関数を利用することも可能である。 CSSを利用することで、背景の色を変更したり、文字の色を変えたり、アンダーラインを引いたりなど多くのことが可能になった。 またWeb開発ツールを利用した、CSSで作成したページのデバックの仕方を理解した。

演習ではWebブラウザの紹介をするCSSファイルを編集し、参考画像と同じウェブページに編集行った。 またindex.htmlとproject課題の装飾も行った。
